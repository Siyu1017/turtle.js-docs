# Methods

### Register a plugin
```js copy
t.register(plugin)
```

### Unregister a plugin
```js copy
t.unregister(pluginName)
```

### Enable plugin
```js copy
t.enable(pluginName)
```

### Disable plugin
```js copy
t.disable(pluginName)
```

### Toggle enable plugin
```js copy
t.toggle(pluginName)
```
