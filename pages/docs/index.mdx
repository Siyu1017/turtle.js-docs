# Get started

Turtle.js is a javaScript library for implementing turtle graphics in non-Python environments, it can be used in any web applications.

## Install

To install Turtle.js, you can use a package manager like npm or a CDN.

### NPM
```bash copy
npm install @siyu971017/turtle.js
```

## CDN

Turtle.js can be loaded via CDN using ESM or UMD format.

### ESM
```js copy
import turtle from "https://cdn.jsdelivr.net/npm/@siyu971017/turtle.js@1.2.1/+esm";
```

### UMD
```html copy
<script src="https://cdn.jsdelivr.net/npm/@siyu971017/turtle.js@1.2.1/dist/turtle.umd.min.js"></script>
```
All exports will be available on `window.turtle`.
